Графический калькулятор (версия 1.0)

- запустиете файл CalculatorGraphic.jar (без использования консоли)
- введите в текстовое поле первое число при помощи кнопок;
- выберите оператор;
- введите в текстовое поле второе число при помощи кнопок;
- нажмите на "=" (обязательно, для получения результата, что бы программа зафиксировала результат) // исправить в будующих версиях;
- при необходимости выберите оператор для продолжения расчетов;
